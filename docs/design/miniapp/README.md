# VPN Telegram Mini App — handoff для Claude Code

Здесь всё, чтобы воспроизвести дизайн **1 в 1**: рабочий исходник, зависимости и текстовая спека.

## Что в папке
- **`VPN Miniapp — 3 варианта.dc.html`** — исходник со ВСЕМИ 8 вариантами UI (1a–1c, 2a–2e). Это и есть 1:1 источник разметки, стилей и логики (React-класс внутри).
- **`ios-frame.jsx`** — рамка iPhone (статус-бар, dynamic island). Подключается из исходника.
- **`support.js`** — рантайм Design Component (рендерит `.dc.html`). Не редактировать.
- **`VPN-miniapp-prompt.md`** — полная спека: экраны, функции, данные, флоу оплаты и подключения (Remnawave), тарифы, реф-программа, описания всех 8 тем.

## Как открыть исходник как есть
Открыть `VPN Miniapp — 3 варианта.dc.html` в браузере через локальный сервер (нужен из-за загрузки соседних файлов):
```
cd handoff && python3 -m http.server 8080
# затем http://localhost:8080/VPN%20Miniapp%20%E2%80%94%203%20варианта.dc.html
```
Внутри: переключатели темы (светлая/тёмная) и языка (RU/EN), кликабельные вкладки, тарифы, шаги подключения.

## Промпт для Claude Code (воспроизвести 1:1)
> Возьми `VPN Miniapp — 3 варианта.dc.html` как эталон дизайна. Пересобери его как продакшн Telegram Mini App на [твой стек: React + Vite + @twa-dev/sdk / другое]. Требования:
> - Сохрани 1:1 верстку, отступы, цвета, шрифты и тексты (RU+EN) каждого из вариантов — выбери со мной один из 8 или собери переключатель тем.
> - Три вкладки: Главная, Подключение, Кабинет (см. `VPN-miniapp-prompt.md`, разделы 2–6).
> - Светлая/тёмная тема по теме Telegram; язык по `language_code`.
> - Вкладка «Подключение» — инструкция из 3 шагов: определить платформу и дать ссылку на стор → запросить персональную subscription-ссылку у Remnawave → deep-link «Открыть в приложении» + «Скопировать». Мини-апп сам VPN НЕ включает.
> - Оплата: Telegram Stars (нативный invoice), карта/СБП (провайдер), USDT. Тарифы и модель данных — из спеки.
> - Реферальная программа: +7 дней обоим.
> Данные вынеси в моки-объекты (User/Subscription/Plan/Payment/Referral/Connection), тексты — в i18n-словарь ru/en.
