# VPN Telegram Mini App — спецификация для конструктора

Промпт-документ для сборки Telegram Mini App по продаже VPN. Содержит: общую логику, список экранов, функции, данные, платёжный и подключательный флоу (Remnawave), плюс 8 визуальных тем на выбор. Копируйте нужные разделы в конструктор.

---

## 1. Общий промпт (вставить целиком)

> Собери Telegram Mini App для продажи VPN-подписок. Приложение НЕ подключается к VPN само — оно продаёт доступ и выдаёт инструкцию с персональной ссылкой. Сам VPN работает во внешнем клиентском приложении.
>
> **Три вкладки в нижней навигации: «Главная», «Подключение», «Кабинет».** Главное действие — статус подписки и её продление.
>
> Поддержи светлую и тёмную темы (переключение по теме Telegram), русский и английский языки (по `language_code` пользователя). Аудитория — обычные люди, интерфейс максимально понятный: минимум терминов, крупные кнопки (тап-таргет ≥ 44px), каждый шаг подписан словами.
>
> Способы оплаты: **Telegram Stars, банковская карта, СБП, криптовалюта (USDT TRC-20)**. Тарифы: 1 мес — 199 ₽, 3 мес — 499 ₽ (−16%), 12 мес — 1 490 ₽ (−38%). Есть реферальная программа: пригласи друга → +7 дней обоим.
>
> Данные пользователя (подписка, дни, устройства, история платежей) берутся из бэкенда; персональная ссылка доступа генерируется панелью **Remnawave**.

## 2. Экраны и функции

### 2.1. Главная (Home)
Назначение: показать статус подписки и дать продлить.

- **Карточка статуса:** индикатор «Подписка активна» (зелёная точка), крупное число оставшихся дней, дата окончания, прогресс-бар (дни/90).
- **Кнопка «Продлить»** — основное действие, открывает оплату выбранного тарифа.
- **Выбор тарифа:** 1 / 3 / 12 месяцев с ценой и скидкой; выделен рекомендуемый (3 мес).
- **Способы оплаты:** чипы СБП / Карта / USDT / Stars.
- **Реф-блок:** «Пригласи друга — +7 дней каждому» → на экран реферала.

Состояние: `subscription.active`, `subscription.daysLeft`, `subscription.expiresAt`, `selectedPlan`.

### 2.2. Подключение (Connect) — инструкция, НЕ тумблер
Пошаговый гайд. Никакой кнопки «включить VPN» внутри мини-аппа.

- **Шаг 1 — Скачать приложение.** Устройство определяется автоматически (`navigator.userAgent` / Telegram platform). Кнопка ведёт в нужный стор: iOS → App Store, Android → Google Play/APK, macOS/Windows → сайт. Подпись: «iOS · Android · macOS · Windows».
- **Шаг 2 — Получить персональную ссылку.** Кнопка «Получить ссылку» запрашивает у Remnawave subscription-URL для пользователя. После ответа показываются:
  - сама ссылка (моно, с обрезкой),
  - кнопка **«Открыть в приложении»** (deep-link `<scheme>://import/<sub_url>` — открывает клиент и импортирует конфиг автоматически),
  - кнопка **«Скопировать»**.
- **Шаг 3 — В приложении нажать «Подключить».** Финальная подсказка.

Функции: `detectPlatform()`, `getSubscriptionLink()` (Remnawave API), `buildDeepLink(client, subUrl)`, `copyToClipboard()`.

### 2.3. Кабинет (Account)
- Профиль: имя, @username, аватар (из Telegram).
- Подписка: до какого числа.
- Устройства: N из 5 (лимит одновременных).
- История платежей (дата · способ · сумма).
- Реферальная программа (+7 дней).
- Поддержка (чат/бот).
- Версия и протокол (WireGuard) — мелко, внизу.

## 4. Флоу оплаты
1. Пользователь выбирает тариф → жмёт «Продлить».
2. Выбор способа: Stars (нативный Telegram invoice), Карта/СБП (провайдер, напр. ЮKassa), USDT (крипто-инвойс).
3. После успешной оплаты бэкенд продлевает подписку → обновляются `daysLeft` / `expiresAt`.
4. Возврат на Главную с обновлённым статусом.

## 5. Флоу подключения (Remnawave)
1. Экран «Подключение» → «Получить ссылку».
2. Бэкенд запрашивает у Remnawave персональный subscription-URL пользователя.
3. Мини-апп показывает ссылку + deep-link + «Скопировать».
4. «Открыть в приложении» → системный клиент импортирует конфиг по ссылке.
5. Пользователь жмёт «Подключить» уже в клиенте.

## 6. Реферальная программа
- У каждого пользователя реф-код/ссылка.
- Пришедший по ссылке друг активирует подписку → обоим +7 дней.
- Показывать в Кабинете и блоком на Главной.

## 7. Визуальные темы (8 вариантов)

Одинаковый набор экранов и функций — отличается только оформление, шрифты и тон копирайтинга. Выберите одну или смешайте.

**1a «Минимал»** — светлый, чистый; системный шрифт (SF/system-ui); один синий акцент `#2E63E7`; нейтральный серо-белый фон; спокойный деловой тон. Кнопка подключения — большое кольцо (в новой версии заменено на инструкцию).

**1b «Прайват»** — строгий «банковский»; серифные цифры (Instrument Serif); тёмно-зелёный `#1E4B39` + золото `#A5824A`; кремовый фон; карта-«пластик» с номером; формальный тон («Добрый день, Алексей», «действует до…»). Подключение — нумерованные шаги 1-2-3, таблица серверов с пингом/нагрузкой.

**1c «Бадди»** — тёплый, дружелюбный; шрифт Nunito; оранжевый `#EF7048` + мятный `#3FBF97`; маскот-персонаж с репликами («Привет! Ты под защитой», «Я на страже!»); округлые формы, пилюли; разговорный тон.

**2a «Нативный»** — как продолжение Telegram: системные сгруппированные списки, голубой акцент `#2AA0DC`, серый фон iOS `#EFEFF4`; ощущение родного раздела Telegram.

**2b «Терминал»** — моноширинный (JetBrains Mono), техно-стиль для гиков; зелёный `#0C8F4E` на светлом / неон в тёмной теме; нижний регистр, `01/02/03`, `d`-дни, `wg · aes-256`. Хорошо смотрится в тёмной теме.

**2c «Журнал»** — editorial/печатный; сериф (Instrument Serif); кремовый фон, красный акцент `#C03B2D`; линейки-разделители вместо карточек, крупные порядковые «1. 2. 3.»; тон как в журнальной статье.

**2d «Неон»** — глассморфизм; фиолетово-бирюзовый градиент `#7C5CFF → #1FC7E0`; полупрозрачные размытые карточки, свечение кнопок; современный «премиум-технологичный» тон. Ярче всего в тёмной теме.

**2e «Поп»** — дуотон лайм `#C8F04B` + чёрный; толстые обводки 2px, жёсткие тени-оффсеты, стикеры с наклоном, гротеск Unbounded/Archivo Black; кричащий тон капсом («ДВА ШАГА», «КАЧАЙ ПРИЛОЖЕНИЕ»). Молодёжный/мемный.

### Как задать тему в промпте
> Оформи в стиле **[название]**: палитра [...], шрифт [...], тон [...]. Экраны и функции — из разделов 2–6.

## 8. Тон копирайтинга (RU) по темам
- Минимал / Нативный: нейтрально-деловой. «Подписка активна», «Продлить».
- Прайват / Журнал: формально. «Действует до…», «Активируйте подписку».
- Бадди / Поп: неформально. «Ещё 23 дня защиты», «Дай ссылку!».

Английские строки — параллельно, тот же смысл. Все тексты вынести в i18n-словарь `ru/en`.
