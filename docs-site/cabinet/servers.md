# Серверы

Экран «Серверы» — локальное зеркало нод и сквадов панели Remnawave: нагрузка и статусы всегда под рукой, без захода в панель. Здесь же ноды снимаются с продажи, а пользователям в боте можно включить публичную страницу статуса.

## Синхронизация с панелью

Зеркало обновляется двумя путями:

- **автоматически** — фоновая задача каждые 15 минут забирает ноды из панели;
- **вручную** — кнопка «Синхронизировать» на экране (полезно сразу после изменений в панели).

Как устроена связка бота с панелью в целом — см. [Синхронизация с панелью](/panel/sync).

По каждой ноде синкаются: имя, код страны, адрес, число онлайн-пользователей, трафик за день и статус. Вместе с нодами зеркалируются внутренние сквады панели: их имя и число участников. Панельное имя сквада хранится отдельно от отображаемого — переименование в кабинете переживает синк.

::: tip Пропавшие ноды не удаляются
Нода, исчезнувшая из панели, помечается OFFLINE, но остаётся в списке — её локальные флаги (например, «в продаже») переживают сбои и переезды панели.
:::

## Статусы нод

| Статус | Значок в боте | Когда |
|---|---|---|
| `online` | 🟢 | Нода в панели онлайн |
| `maintenance` | 🟠 | Нода отключена в панели (disabled) |
| `offline` | 🔴 | Нода не отвечает или пропала из панели |

В таблице кабинета по каждой ноде видны: статус, страна, адрес, онлайн, трафик за день, нагрузка (%), пинг, аптайм (%), флаг «в продаже» и время последнего синка.

## Флаг «в продаже»

`is_for_sale` — флаг бота, а не панели (по умолчанию включён). Выключение снимает сквады ноды с продажи: новые покупки на неё не назначаются, при этом существующие пользователи панели не трогаются и продолжают работать. Удобно при переполнении ноды или плановых работах — выключили продажу, дождались разгрузки, включили обратно.

Переключение фиксируется в журнале аудита.

## Статус серверов пользователю в боте

Пользователям можно показать живой статус нод — кнопка «Статус серверов» в боте выводит список:

```text
🌍 Статус серверов

🟢 NL Amsterdam-1 · онлайн 213
🟠 DE Frankfurt-2 · онлайн 0
🔴 FI Helsinki-1 · онлайн 0
```

Формат строки: значок статуса, код страны, имя ноды, текущий онлайн. Показываются до 30 нод, отсортированных по имени. Пока зеркало пустое (первый синк ещё не прошёл), пользователь видит «Данные ещё собираются».

Раздел включается параметром в [Настройках](/cabinet/settings):

| Параметр | По умолчанию | Что это |
|---|---|---|
| `NODE_STATUS_ENABLED` | выкл | Кнопка «Статус серверов» в боте: 🟢/🟠/🔴 по нодам с онлайном |

::: info Зеркало кормит и другие фичи
Список ONLINE-нод из этого зеркала использует [Контроль устройств](/features/device-guard) — сканирование онлайн-IP идёт только по живым нодам.
:::
