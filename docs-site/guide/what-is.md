# Что это

**VPN-HUB BOT** — конструктор Telegram-ботов для продажи VPN на базе панели [Remnawave](https://remna.st). Принимает оплату, выдаёт подписки, управляет пользователями — а весь бот собирается и настраивается через веб-кабинет, без правки кода и перезапусков.

Платформа состоит из четырёх «лиц» на одном общем бэкенд-ядре:

| Лицо | Что делает | URL |
|---|---|---|
| **Telegram-бот** | продажи, триал, баланс, рефералка, тикеты; меню собирается в конструкторе кабинета | — |
| **Админ-кабинет** | React-SPA: 15 экранов, тарифы, платёжки, рассылки, пользователи, настройки | `/admin/` |
| **Mini App** | личный кабинет подписчика внутри Telegram, 8 визуальных тем | `/app/` |
| **Бэкенд** | FastAPI + фоновые задачи (taskiq) + бизнес-ядро — один процесс-граф на всех | `/api/…` |

Один и тот же контейнер зависимостей (`AppContainer`) переиспользуется в web, боте и воркере — бизнес-логика везде одна.

![Админ-кабинет — дашборд](/assets/admin-dashboard.png)

## Для кого

Для владельцев VPN-магазинов на Remnawave, которые продают, а не программируют. Всё, что видит юзер — кнопки бота, темы мини-аппы, тарифы, акции — собирается мышкой в кабинете и применяется сразу, hot-reload без рестартов. Если вы уже продаёте через remnawave-shopbot, Bedolaga, RemnaShop или 3x-ui — есть [миграция](/guide/migration) с переносом юзеров, подписок и ключей панели.

## Что умеет из коробки

**[Бот](/bot/):** покупка (баланс / Telegram Stars / карта, СБП, крипта), триал с лимитами, смена тарифа с зачётом остатка дней, докупка гигабайт, рефералка с выводом заработка (карта/USDT/TON), gift-коды по диплинку, гейт подписки на каналы, устройства HWID с самостоятельной отвязкой, тикеты, режим техработ.

**[Кабинет](/cabinet/):** дашборд с выручкой и онлайном, пользователи с drawer-карточкой, конструктор цен (периоды + пакеты трафика), промокоды и пачки gift-кодов с CSV, рассылки с живым прогрессом, умные напоминания о продлении, атрибуция рекламных кампаний, серверы Remnawave (синк, нагрузка, «в продаже»), 70+ настроек с hot-reload — секреты шифруются.

**[Mini App](/miniapp/):** три вкладки (статус и оплата, подключение в 3 шага с deep-link в happ/v2raytun/hiddify/streisand, кабинет), 8 тем на выбор владельца, светлая/тёмная по теме Telegram.

**[Платежи](/payments/):** 21 живой провайдер за одним ABC — от Telegram Stars и YooKassa до крипты (CryptoBot, Cryptomus, Heleket). Подписка выдаётся автоматически по вебхуку; фоновый реконсилятор каждые 5 минут добивает зависшие оплаты, если вебхук потерялся. Возвраты — из кабинета.

**[Панель](/panel/):** panel-first провижининг, ночная сверка бот↔панель, вебхук событий, watchdog — панель упала, бот сам включает заглушку техработ и снимает её, когда панель поднялась. Нет живой панели — в комплекте [мок-панель](/panel/mock) с теми же эндпоинтами, переключение real ↔ mock — один env.

Плюс: покупка с сайта без Telegram (email / Google / Yandex), Device Guard (детект шеринга по онлайн-IP), шифрованные бэкапы БД, ИИ-поддержка в тикетах.

## Стек

Python 3.12 async · aiogram 3 · FastAPI · SQLAlchemy 2 · PostgreSQL · Redis · taskiq — и React 18 + TypeScript для кабинета. Мини-аппа — ваниль HTML/CSS/JS без сборки. Ядро — чистая архитектура в 4 кольца, mypy strict, детали — в разделе [Архитектура](/dev/architecture). Лицензия MIT.
